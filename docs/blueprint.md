# **App Name**: ChargeAssist

## Core Features:

- User Authentication: Implement login and signup pages for users and admins using temporary email authentication.
- Charging Station Locator: Display available charging stations on a map, showing real-time location and availability.
- Station Details Page: Show detailed information for each charging station, including the number of available slots and station details.
- Booking Form: Allow users to book a charging slot by providing their name, phone number, and vehicle number.
- Emergency Charging Service Request: Enable users to request an emergency charging service at their location via a detailed form.
- Admin Dashboard: Provide an admin dashboard to manage charging stations, user details, payment information, approve/deny emergency service requests, and manage slot availability.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF), evoking innovation and energy.
- Background color: Dark gray (#28282B), providing contrast and a modern feel.
- Accent color: Vibrant green (#00FF00), indicating availability and calls to action.
- Body: 'Inter' sans-serif font for a clean and modern look. Headlines: 'Space Grotesk' for impactful titles.
- Use clean, modern line icons to represent charging stations, slot availability, and service types.
- Design a user-friendly layout with intuitive navigation and clear information hierarchy, prioritizing ease of use.
- Incorporate subtle animations and transitions to enhance user engagement and provide feedback on interactions.
